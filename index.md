# Text Selection

```css
html body :is(h1, h2, h3, h4, h5, h6, p, span, a, li, dt, dd,
          blockquote, q, pre, figcaption, caption, th, td) { user-select: text
          !important; /* Chrome, Opera, Edge, Cốc Cốc */ -webkit-user-select:
          text !important; /* Safari */ -khtml-user-select: text !important; /*
          Konqueror */ -moz-user-select: text !important; /* Firefox */ }
```
